Gammapy tutorial notebooks¶

What is this?¶

  • This is an overview of tutorial Jupyter notebooks for Gammapy, a Python package for gamma-ray astronomy.
  • The notebooks complement the Gammapy Sphinx-based documentation at http://docs.gammapy.org
  • The notebooks and example datasets are available at https://github.com/gammapy/gammapy-extra

Set up¶

If you want to execute the notebooks locally, to play around with the examples, or to try to do one of the exercises, you have to first install Gammapy and get the gammapy-extra repository. This is described in Gammapy tutorial setup.

You can also download for each version of Gammapy a HTMLZip pack containing the whole documentation and full collection of notebooks, so you can execute them in your local desktop inside the _static/notebooks/ folder.

Alternatively you may execute latest version of the collection of notebooks on-line accessing the Gammapy Binder space.

The basics¶

Gammapy is a Python package built on Numpy and Astropy, and the tutorials are Jupyter notebooks. If you’re already familar with those, you can skip to the next section and start learning about Gammapy.

To learn the basics, here are a few good resources.
