add_pulsar_parameters¶

gammapy.astro.population.add_pulsar_parameters(table, B_mean=12.05, B_stdv=0.55, P_mean=0.3, P_stdv=0.15)[source]¶

Adds pulsar parameters to the table.

For the initial normal distribution of period and logB can exist the following Parameters: B_mean=12.05[log Gauss], B_stdv=0.55, P_mean=0.3[s], P_stdv=0.15
