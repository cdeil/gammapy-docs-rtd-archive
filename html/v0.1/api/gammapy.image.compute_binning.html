compute_binning¶

gammapy.image.compute_binning(data, n_bins, method='equal width', eps=1e-10)[source]¶

Computes 1D array of bin edges.

The range of the bin_edges is always [min(data), max(data)]

Note that bin_edges has n_bins bins, i.e. length n_bins + 1.

Parameters:

data : array_like

Data to be binned (any dimension)

n_bins : int

Number of bins

method : str

One of: ‘equal width’, ‘equal entries’

eps : float

added to range so that the max data point is inside the last bin. If eps=0 it falls on the right edge of the last data point and thus would be not cointained.
Returns:

bin_edges : 1D ndarray

Array of bin edges.
