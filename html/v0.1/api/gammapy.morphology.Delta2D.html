Delta2D¶

class gammapy.morphology.Delta2D(amplitude, x_0, y_0, **constraints)[source]¶

Bases: astropy.modeling.core.Fittable2DModel

Two dimensional delta function .

This model can be used for a point source morphology.

Parameters:

amplitude : float

Peak value of the point source

x_0 : float

x position center of the point source

y_0 : float

y position center of the point source

See also

Shell2D, Sphere2D, astropy.modeling.models.Gaussian2D

Notes

Model formula:

The pixel positions x_0 and y_0 are rounded to integers. Subpixel information is lost.
