Data classes (gammapy.data)¶

Introduction¶

gammapy.data contains classes to represent gamma-ray data.

An EventList contains unbinned data, i.e. the longitude, latitude, energy and time for each event.

A SpectralCube contains binned dat as a 3-dimensional array with longitude, latitude and spectral axes. The event time information has been removed ... it is represented by a GTI (GTI = Good Time Interval) class that is needed for exposure and flux computations.
