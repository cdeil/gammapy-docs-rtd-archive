XML file format¶

GammaLib / ctools uses an “model definition” XML format described here

The Fermi gtlike tool uses the same format (the implemented models are a bit different) described here

The same format is used by ctlike and gtlike to report the model fit results, which has the serious draw-back that useful information like the fit covariance matrix, asymmetric errors or numbers like the fit statistic or predicted number of counts are not output in a machine-readable format.
