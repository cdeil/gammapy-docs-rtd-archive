Functions¶

create_scale_space(image, scales[, kernel]) Creates Scale Space for a given image and stores it in 3D array.
detect_blobs(image_3D, scales, threshold) Detect blobs of different sizes.
detect_blobs_3D(image, threshold) Find maxima in image cubes.
detect_peaks(image) Detect peaks in an image using a maximum filter.
detect_peaks_3D(image) Same functionality as detect_peaks, but works on image cubes.
probability_center(images, kernel) Compute matched-filter p-value at the kernel center.
probability_image(images, kernel) Compute matched-filter p-value image.
prune_blobs(blobs, overlap_threshold, q_factor) Prune blobs.
read_scale_cube(filename) Load filtered images from numpy binary file.
read_scale_cube_fits(filename) Read scale space from fits file image cube.
run_detection(args) Run iterative source detection.
sex(image[, image2, catalog_name, ...]) Run SExtractor to detect sources in an image.
show_blobs(image, blobs) Show input image with overlaid blobs.
show_peaks(image_3D) Show all images of different scales including the detected peaks.
significance_center(images, kernel) Compute matched-filter significance at the kernel center.
significance_image(images, kernel) Compute matched-filter significance image.
ts_image(images, kernel[, extra_info]) Compute TS image.
write_region_file(regionfile, blobs) Write ds9 region file from blob list.
