Gammapy is an open source (BSD licensed) gamma-ray astronomy Python package.

It is an in-development affiliated package of Astropy that builds on the core scientific Python stack to provide tools to simulate and analyse the gamma-ray sky for telescopes such as Fermi, H.E.S.S. and CTA.

Currently the functionality is limited and geared towards Galactic plane survey studies, but the idea is that Gammapy is a place to share and collaborate for gamma-ray astronomy Pythonistas ... contributions welcome!

  • Read the Gammapy documentation.
  • Ask questions on the Gammapy mailing list.
  • Request features, report bugs or contribute on the Gammapy GitHub page.
  • Check out the Gammapy project summary on Ohloh or the Gammapy page on PyPI.
  • Requires Python (version 2.7 and 3.3 or later), Numpy (version 1.6 or later) and Astropy (version 0.4 or later)

The Gammapy toolbox¶

  • Astrophysical source and population models (gammapy.astro)
  • Background estimation and modeling (gammapy.background)
  • Catalog (gammapy.catalog)
  • Data classes (gammapy.data)
  • Access datasets (gammapy.datasets)
  • Source detection tools (gammapy.detect)
  • Image processing and analysis tools (gammapy.image)
  • Instrument response function (IRF) functionality (gammapy.irf)
  • Morphology and PSF methods (gammapy.morphology)
  • Observation bookkeeping (gammapy.obs)
  • Air shower image I/O and analysis (gammapy.shower)
  • Spectrum estimation and modeling (gammapy.spectrum)
  • Statistics tools (gammapy.stats)
  • Utility functions and classes (gammapy.utils)
