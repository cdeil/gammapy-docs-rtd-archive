Maps¶

class gammapy.background.Maps(hdus=, []file=None, rename_hdus=None, is_off_correlated=True, theta=None, theta_pix=0)[source]¶

Bases: astropy.io.fits.hdu.hdulist.HDUList

Maps container for basic maps and methods to compute derived maps.

It is simply a list of HDUs containing the maps, plus methods to compute the derived maps.

These maps allow implementing all background estimation methods. Not all maps are used for each method, unused maps are typically filled with zeros or ones as appropriate.
