ra_iau_format¶

gammapy.catalog.ra_iau_format(ra, digits)[source]¶

Right Ascension part of an IAU source designation.

Reference: http://cdsweb.u-strasbg.fr/Dic/iau-spec.html

digits format
2 HH
3 HHh
4 HHMM
5 HHMM.m
6 HHMMSS
7 HHMMSS.s
Parameters:

ra : Longitude

Right ascension

digits : int (>=2)

Number of digits
Returns:

strrepr : str

IAU format string representation of the angle
