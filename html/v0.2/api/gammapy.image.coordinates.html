coordinates¶

gammapy.image.coordinates(image, world=True, lon_sym=True, radians=False)[source]¶

Get coordinate images for a given image.

This function is useful if you want to compute an image with values that are a function of position.

Parameters:

image : ImageHDU or ndarray

Input image

world : bool, optional

Use world coordinates (or pixel coordinates)?

lon_sym : bool, optional

Use symmetric longitude range (-180, 180) (or (0, 360))?

radians : bool, optional

Return coordinates in radians or degrees?
Returns:

(lon, lat) : tuple of arrays

Images as numpy arrays with values containing the position of the given pixel.

Examples

>>> import numpy as np
>>> from gammapy.datasets import FermiGalacticCenter
>>> lon, lat = coordinates(FermiGalacticCenter.counts())
>>> dist = np.sqrt(lon ** 2 + lat ** 2)
