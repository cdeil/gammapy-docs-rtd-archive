Bases: astropy.table.Table

Data store index table.

The index table is a FITS file that stores which observations are available and what their most important parameters are.

This makes it possible to select observation of interest and find out what data is available without opening up thousands of FITS files that contain the event list and IRFs and have similar information in the FITS header.
