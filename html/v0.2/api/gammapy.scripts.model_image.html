model_image¶

gammapy.scripts.model_image()[source]¶

Given a catalog of sources, simulate a flux image.

Inputs:

  • Source list (JSON file)
  • PSF (JSON file)
  • Exposure image (FITS file)

Outputs:

  • Source model flux image (FITS file)
  • Source model excess image (FITS file)
