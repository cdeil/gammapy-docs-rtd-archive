Galprop¶

class gammapy.spectrum.Galprop[source]¶

Bases: object

A cylindrically symmetric model for the distribution of the optical and infrared ISRF in the Milky Way.

The CMB component is isotropic, so it’s implementation is different from the two non-isotropic components.

http://vizier.cfa.harvard.edu/vizier/ftp/cats/J/A+A/534/A54/galprop_v54.pdf
