TableModel¶

class gammapy.spectrum.models.TableModel(e, y, emin=None, emax=None)[source]¶

Bases: gammapy.spectrum.models.Model

A spectrum represented by numeric arrays of x and y values.

Internally all calculations are done on log10(x) and log10(y) for numerical stability and accuracy.
