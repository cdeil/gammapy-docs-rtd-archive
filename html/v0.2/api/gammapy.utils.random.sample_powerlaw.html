sample_powerlaw¶

gammapy.utils.random.sample_powerlaw(x_min, x_max, gamma, size=None)[source]¶

Sample random values from a power law distribution.

f(x) = x ** (-gamma) in the range x_min to x_max

Reference: http://mathworld.wolfram.com/RandomNumber.html

Parameters:

x_min : float

x range minimum

x_max : float

x range maximum

gamma : float

Power law index

size : int

Number of samples to generate
Returns:

x : array

Array of samples from the distribution
