SimplePulsar¶

class gammapy.astro.source.SimplePulsar(P, P_dot, I=<Quantity 1e+45 cm2 g>, R=<Quantity 1000000.0 cm>)[source]¶

Bases: object

Magnetic dipole spin-down model for a pulsar.

Reference: http://www.cv.nrao.edu/course/astr534/Pulsars.html

Parameters:

P : Quantity

Rotation period (sec)

P_dot : Quantity

Rotation period derivative (sec sec^-1)

I : Quantity

Moment of inertia (g cm^2)

R : Quantity

Radius of the pulsar (cm)

Attributes Summary

luminosity_spindown Spin-down luminosity (Quantity)
magnetic_field Magnetic field strength at the polar cap (Quantity)
tau Characteristic age (Quantity)

Attributes Documentation

luminosity_spindown¶

Spin-down luminosity (Quantity)

Notes

The spin-down luminosity is given by:

\[\dot{L} = 4\pi^2 I \frac{\dot{P}}{P^{3}}\]
magnetic_field¶

Magnetic field strength at the polar cap (Quantity)

Notes

The magnetic field is given by:

\[B = 3.2\cdot 10^{19} (P\dot{P})^{1/2} [\textnormal(Gauss)]\]
tau¶

Characteristic age (Quantity)

Notes

The characteristic age is given by:

\[\tau = \frac{P}{2\dot{P}}\]
