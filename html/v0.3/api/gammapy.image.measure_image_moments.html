measure_image_moments¶

gammapy.image.measure_image_moments(image)[source]¶

Compute 0th, 1st and 2nd moments of an image.

NaN values are ignored in the computation.

Parameters:

image :`astropy.io.fits.ImageHDU`

Image to measure on.
Returns:

image moments : list

List of image moments: [A, x_cms, y_cms, x_sigma, y_sigma, sqrt(x_sigma * y_sigma)]
