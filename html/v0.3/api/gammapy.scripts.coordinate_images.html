coordinate_images¶

gammapy.scripts.coordinate_images(infile, outfile, make_coordinate_maps, make_distance_map, overwrite)[source]¶

Make maps that can be used to create profiles.

The following images can be created: * LON – Longitude coordinate * LAT – Latitude coordinate * DIST – Distance to mask * SOLID_ANGLE – Solid angle
