Schlickeiser¶

class gammapy.spectrum.Schlickeiser[source]¶

Bases: object

ISRF model for the solar neighborhood.

Reference: Book by Schlickeiser “Cosmic Ray Astrophysics”, Section 2.3

Note: component 1 (spectral type B) from Schlickeiser is not very important and was ignored so that there is a 1:1 match to the three Galprop ISRF components.

Note: the form of the ISRF component spectra is like a blackbody, which is characterized by only one number, the temperature T. But the energy density of the ISRF infrared component is different than the corresponding energy density of the blackbody, so each component has one more energy density parameter W and is called a “greybody” distribution by Schlickeiser.
