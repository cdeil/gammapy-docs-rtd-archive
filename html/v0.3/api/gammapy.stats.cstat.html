cstat¶

gammapy.stats.cstat(n_observed, mu_observed, n_observed_min=1e-25)[source]¶

C statistic, for Poisson data.

The C statistic is defined as

\[C = 2 \left[ \mu_{observed} - n_{observed} + n_{observed} (\log(n_{observed}) - log(\mu_{observed}) \right]\]

and \(C = 0\) where \(\mu_{observed} <= 0\).
