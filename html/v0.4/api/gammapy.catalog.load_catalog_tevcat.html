load_catalog_tevcat¶

gammapy.catalog.load_catalog_tevcat()[source]¶

Load TeVCat source catalog.

TeVCat is an online catalog for TeV astronomy.

Unfortunately the TeVCat is not available in electronic format.

This is a dump of TeVCat as of 2014-09-24 created by scraping their web page using the script available here.
