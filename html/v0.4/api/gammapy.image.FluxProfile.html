FluxProfile¶

class gammapy.image.FluxProfile(x_image, x_edges, counts, background, exposure, mask=None)[source]¶

Bases: object

Flux profile.

Note: over- and underflow is ignored and not stored in the profile

Note: this is implemented by creating bin labels and storing the input 2D data in 1D pandas.DataFrame tables. The 1D profile is also stored as a pandas.DataFrame and computed using the fast and flexible pandas groupby and apply functions.
