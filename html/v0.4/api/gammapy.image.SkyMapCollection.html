SkyMapCollection¶

class gammapy.image.SkyMapCollection(name=None, wcs=None, meta=None, **kwargs)[source]¶

Bases: gammapy.extern.bunch.Munch

Container for a collection of sky maps.

This class bundles as set of SkyMaps in single data container and provides convenience methods for Fits I/O and Bunch like handling of the data members.

Here’s an example how to use it:

from gammapy.image import SkyMapCollection
skymaps = SkyMapCollection.read('$GAMMAPY_EXTRA/datasets/fermi_survey/all.fits.gz')

Then try tab completion on the skymaps object.
