dict_to_hdulist¶

gammapy.image.dict_to_hdulist(image_dict, header)[source]¶

Take a dictionary of image data and a header to create a HDUList.

Parameters:

image_dict : dict

Dictionary of input data. The keys are used as FITS extension names. Image data are the corresponding values.

header : astropy.io.fits.Header

Header to be used for all images.
Returns:

hdu_list : astropy.io.fits.HDUList

HDU list of input dictionary.
