image_ts¶

gammapy.scripts.image_ts(input_file, output_file, psf, model, scales, downsample, residual, morphology, width, overwrite, threshold)[source]¶

Compute source model residual images.

The input data FITS file must contain the following HDU extensions:

  • ‘counts’ – Counts image
  • ‘background’ – Background image
  • ‘exposure’ – Exposure image
