SED¶

class gammapy.spectrum.SED[source]¶

Bases: list

Class to plot GeV – TeV SEDs

Internally the same units as in the Fermi catalog are used: - Energies in MeV - Flux densities in cm^-2 s^-2 MeV^-1 - Fluxes in cm^-2 s^-1 - Energy fluxes in erg cm^-2 s^-1

Read necessary parameters from FITS file and plot butterfly

Parameters: catalog_format = ‘hess’, ‘fermi’ catalog_name = FITS file name object_name = object name string in ‘name’ column

Note: Since every catalog has columns with different names and units, a general SED plotting is not possible. Instead for each catalog type a handler function that deals converts to a standard format is called.
