generate_MC_data¶

gammapy.spectrum.generate_MC_data(model, xlim, yerr, npoints=5, verbose=0, random_state=u'random-seed')[source]¶

Generate points with equal-log spacing in x given by the model.

Parameters:

model = [function, parameters, constants]

xlim = [xmin, xmax]

yerr = [ydn_err, yup_err] = fractional error on model y
Returns:

data = x, y, ydn, yup
