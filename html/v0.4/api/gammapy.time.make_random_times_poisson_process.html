make_random_times_poisson_process¶

gammapy.time.make_random_times_poisson_process(size, rate, dead_time=<TimeDelta object: scale='None' format='sec' value=0.0>, random_state=u'random-seed')[source]¶

Make random times assuming a Poisson process.

This function can be used to test event time series, to have a comparison what completely random data looks like.

For the implementation see here and here, as well as numpy.random.exponential.
