galactic_to_radec¶

gammapy.utils.coordinates.galactic_to_radec(glon, glat, unit=u'deg')[source]¶

Convert Galactic to Equatorial J2000 coordinates.

Parameters:

glon, glat : array_like

Galactic coordinates

unit : {‘deg’, ‘rad’}

Units of input and output coordinates
Returns:

ra, dec : array_like

Equatorial coordinates.

Notes

This is a standalone implementation that only uses numpy for testing. Use SkyCoord instead. Only accurate to ~ 3 digits.
