table_to_fits_table¶

gammapy.utils.fits.table_to_fits_table(table)[source]¶

Convert astropy table to binary table FITS format.

This is a generic method to convert a Table to a BinTableHDU. The name of the table can be stored in the Table meta information under the name keyword.

Parameters:

table : Table

astropy table containing the desired columns
Returns:

hdu : BinTableHDU

FITS bin table containing the astropy table columns
