cta_irf_root_to_fits¶

gammapy.utils.pyfact.cta_irf_root_to_fits(irf_root_file_name, write_output=False)[source]¶

Convert CTA IRF data from ROOT to FITS format.

This script converts a CTA response stored in a root file into a set of FITS files, namely ARF, RMF, and one auxiliary file, which stores all information from the response file in simple fits tables.
