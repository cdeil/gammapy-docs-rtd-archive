get_cam_acc¶

gammapy.utils.pyfact.get_cam_acc(camdist, rmax=4.0, nbins=None, exreg=None, fit=False, fitfunc=None, p0=None)[source]¶

Calculates the camera acceptance histogram from a given list with camera distances (event list).

Parameters:

camdist : array

Numpy array of camera distances (event list).

rmax : float, optional

Maximum radius for the acceptance histogram.

nbins : int, optional

Number of bins for the acceptance histogram (default = 0.1 deg).

exreg : array, optional

Array of exclusion regions. Exclusion regions are given by an aray of size 2 [r, d] with r = radius, d = distance to camera center

fit : bool, optional

Fit acceptance histogram (default=False).
