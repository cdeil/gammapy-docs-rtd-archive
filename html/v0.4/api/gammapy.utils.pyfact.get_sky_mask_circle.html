get_sky_mask_circle¶

gammapy.utils.pyfact.get_sky_mask_circle(r, bin_size)[source]¶

Returns a 2d numpy histogram with (2. * r / bin_size) bins per axis where a circle of radius has bins filled 1.s, all other bins are 0.

Parameters:

r : float

Radius of the circle.

bin_size : float

Physical size of the bin, same units as rmin, rmax.
Returns:

sky_mask : 2d numpy array

Returns a 2d numpy histogram with (2. * r / bin_size) bins per axis where a circle of radius has bins filled 1.s, all other bins are 0.
