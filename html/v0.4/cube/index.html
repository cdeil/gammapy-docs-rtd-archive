Cube Style Analysis (gammapy.cube)¶

The cube module bundles functionality for combined spatial and spectral analysis (cube style anlysis) of gamma-ray sources. Some information on cube style analysis in gamma-ray astronomy can be found here:

  • Cube style analysis for Cherenkov telescope data
  • Classical analysis in VHE gamma-ray astronomy

Reference/API¶

gammapy.cube Package¶

Functions¶

convolve_cube(cube, psf, offset_max) Convolves a predicted counts cube in energy bins with the an energy-dependent PSF.
