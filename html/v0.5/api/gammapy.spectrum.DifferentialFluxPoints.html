DifferentialFluxPoints¶

class gammapy.spectrum.DifferentialFluxPoints(data=None, masked=None, names=None, dtype=None, meta=None, copy=True, rows=None, copy_indices=True, **kwargs)[source]¶

Bases: astropy.table.Table

Differential flux points table

Column names: ENERGY, ENERGY_ERR_HI, ENERGY_ERR_LO, DIFF_FLUX, DIFF_FLUX_ERR_HI, DIFF_FLUX_ERR_LO For a complete documentation see SED, for an usage example see Flux point computation.

Upper limits are stored as in the Fermi catalogs. I.e. the lower error is set to NaN, while the upper error represents the 1 sigma upper limit.
