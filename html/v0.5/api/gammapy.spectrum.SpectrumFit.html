SpectrumFit¶

class gammapy.spectrum.SpectrumFit(obs_list, model, stat=u'wstat', method=u'simplex')[source]¶

Bases: object

Spectral Fit using Sherpa

This is a wrapper around Fit that takes care about translating gammapy classes to sherpa classes and handling various aspects of the fitting correctly. For usage examples see Spectral Fitting

Parameters:

obs_list : SpectrumObservationList

Observations to fit

model : SpectralModel, ArithmeticModel

Model to be fit

stat : str, Stat

Fit statistic to be used

method : str, OptMethod

Fit statistic to be used
