SpectrumObservationGrouping¶

class gammapy.spectrum.SpectrumObservationGrouping(obs_table)[source]¶

Bases: object

Class for stacking observations in groups

The format of the input observation table is described in Observation lists. The column PHAFILE is added after a SpectrumExtraction. The column GROUP_ID can be added as described in Observation grouping.

Parameters:

obs_table : SpectrumObservationList

Observation table with group ID column
