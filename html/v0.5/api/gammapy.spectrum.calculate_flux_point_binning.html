calculate_flux_point_binning¶

gammapy.spectrum.calculate_flux_point_binning(obs_list, min_signif)[source]¶

Compute energy binning.

This is useful to get an energy binning to use with compute() Each bin in the resulting energy binning will include a min_signif source detection.
